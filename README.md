===========

Liquid templating for Rust

Usage
----------

To include liquid in your project add the following to your Cargo.toml:

```toml
[dependencies.liquid]

git = "https://github.com/cobalt-org/liquid-rust.git"

```

Now you can use the crate in your code
```rust
extern crate liquid;
```
